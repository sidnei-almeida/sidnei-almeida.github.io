Previsão de Falhas SECOM

Ativo Recente Destaque

Sistema avançado de detecção de anomalias em manufatura de semicondutores utilizando Autoencoder Neural Network com dashboard Streamlit interativo. SECOM Failure Prediction é um sistema de detecção de anomalias desenvolvido para identificar falhas em processos de manufatura de semicondutores. Utiliza Autoencoder Neural Network com arquitetura 558 → 128 → 64 → 32 (bottleneck) → 64 → 128 → 558. O sistema aprende padrões de operação normal e detecta desvios que podem indicar potenciais falhas. Dashboard interativo com Streamlit, dois thresholds de detecção (Balanced 0.45 e Conservative 0.50), visualizações avançadas com Plotly, design dark premium e performance otimizada com TensorFlow CPU. Dataset SECOM: 1567 registros, 558 features, desbalanceamento ~93% Normal vs ~7% Falhas.
