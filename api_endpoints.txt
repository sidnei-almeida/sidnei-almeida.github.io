Base URL
--------
https://keras-imdb-sentiment-lstm.onrender.com/

Available Endpoints
-------------------
GET /        -> Returns API metadata, configured paths, and model summary information.
GET /health  -> Health check confirming that the model and tokenizer are loaded successfully.
POST /predict
  Request Body (JSON): { "text": "Movie review in English" }
  Response: { "sentiment": "positive|negative", "confidence": <float>, "probability": <float> }

Example Requests
----------------
Health Check:
  curl https://keras-imdb-sentiment-lstm.onrender.com/health

Prediction:
  curl -X POST https://keras-imdb-sentiment-lstm.onrender.com/predict \
       -H "Content-Type: application/json" \
       -d '{"text": "This movie was absolutely fantastic!"}'
