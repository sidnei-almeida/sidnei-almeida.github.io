## Purpose

This folder exposes the [Economic Monitor SA](../economic_monitoring_sa/README.md) dashboard under a clean, Portuguese-oriented URL slug (`monitoramento-sulamericano`) for portfolio navigation purposes. All core logic, data handling and styling reside in the `economic_monitoring_sa` project.

---

## Structure

| File | Role |
|------|------|
| `monitoramento-sulamericano.html` | Shell HTML — loads `economic_monitoring_sa.css` and `economic_monitoring_sa.js` |
