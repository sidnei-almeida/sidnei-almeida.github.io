Steam sign-in (OAuth with deep link loadout://auth-callback), portfolio dashboard, CS2 skins and inventory, snapshot simulations to run and compare scenarios, and profile management. REST API backend with FastAPI deployed on Render (skinfolio-backend-v2). Stack: React Native 0.82, TypeScript, React 19, React Navigation, TanStack Query, Fast Image, WebView, SVG, Gesture Handler. Release APK via build-apk.sh; Android and iOS support.
