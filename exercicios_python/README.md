# Análise de Pedidos com Python Básico

Exercício guiado em Python puro para gerar um relatório de pedidos de e-commerce.

## Como usar

1. Use `analise_pedidos_guiado.py` nesta pasta (ou baixe pela página do portfólio).
2. Complete as funções marcadas com `TODO`.
3. Descomente `gerar_relatorio(pedidos)` no final do arquivo.
4. Execute: `python analise_pedidos_guiado.py`

## Página do exercício no portfólio

https://sidnei-almeida.github.io/exercises/analise-pedidos-python

## Entrega sugerida

Publique em um repositório `analise-pedidos-python` com este arquivo e um `README.md` documentando objetivo, execução e aprendizados.
